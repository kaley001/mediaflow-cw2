Azure Cloud-Native Mapping

This page explains how the local MediaFlow prototype maps to the CW2 Azure implementation.

Azure App Service

Hosts the MediaFlow web application frontend and backend.

Azure Blob Storage

Stores uploaded image files as scalable binary blob data.

Cosmos DB

Stores image metadata such as title, description, category and upload date.

Logic Apps REST API

Provides CRUD endpoints for creating, reading, updating and deleting image records.

Azure Functions

Can process uploaded images, generate thumbnails and validate files automatically.

Application Insights

Monitors errors, requests, performance and application health.
